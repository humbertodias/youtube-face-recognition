# Face Recognition

Simple python script for face recognition downloading videos directly from YouTube.

# Install

```
pip install -r requirements.txt
```

# Running

```
python fr.py https://www.youtube.com/watch?v=lWvRf5ZbmZA faces/
```

Press q to exit
